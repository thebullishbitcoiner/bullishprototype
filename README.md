# bullishPrototype

A place to experiment with all the **BULLISH** things in the Bitcoin ecosystem.

## Features

- **Dark Theme**: Complete black background with orange (#ff9900) accent colors
- **Responsive Design**: Mobile-first approach with working hamburger menu
- **Bitcoin Tools**: Various Bitcoin and Lightning Network utilities
- **Nostr Integration**: NDK demo and Nostr wallet connect
- **Payment Widgets**: TwentyUno integration for easy Bitcoin payments

## Pages

- **Home**: Main landing page
- **TwentyUno**: Bitcoin payment widgets
- **Bitcoin Connect**: Bitcoin Connect protocol demo
- **Bitcoin QR**: QR code generation for Bitcoin addresses
- **NDK Demo**: Nostr Development Kit functionality
- **LNURL-verify**: Lightning Network URL verification
- **Nostr**: Nostr protocol tools
- **NWC**: Nostr Wallet Connect

## Dark Theme

The entire website features a comprehensive dark theme with:
- Pure black (#000000) background
- White text (#ffffff)
- Orange accent color (#ff9900) for highlights and interactions
- Dark cards and form elements
- Custom scrollbars and focus states
- Mobile-optimized navigation

## Development

```bash
npm install
npm start
```
